# Security Policy

This repository should contain only public-safe Codex skill material.

## Reporting

If you find a real credential, private file reference, customer data, or other
sensitive material in this repository, please open a private security advisory
or contact the maintainer directly.

## Public Boundary

Do not submit:

- Passwords, API keys, tokens, cookies, SSH keys, or private keys.
- Raw private chats, customer records, contracts, payment screenshots, or ID data.
- Local machine paths that reveal private project structure.
- Internal deployment details for private infrastructure.

When in doubt, describe the reusable workflow in abstract terms and keep private
evidence out of the repository.
